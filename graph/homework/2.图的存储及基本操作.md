   2. 如何判别任意两个顶点i和j是否有边相连

      ```
      无向图中：看A[i][j] 值是否为1

      ```

4. 写出从图的邻接表表示转换成邻接矩阵表示的算法。

   ```c
   void convert(ALGraph *G,int arcs[M][N]){
       for(i=0;i<n;i++){
           p=(G->v[i]).firstarc;
           while(p!=null){
               arcs[i][p->data]=1;
               p=p->nextarc;
           }
       }
   }
   ```
